## 安装 Rust

- 官网：https://www.rust-lang.org/
- Linux or Mac:
  - curl https://rustup.rs -sSf | sh
- Windows: 按官网指示操作
- Windows Subsystem for Linux:
  - curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

## 更新与卸载 Rust

- 更新 Rust
  - rustup update
- 卸载 Rust
  - rustup self uninstall
